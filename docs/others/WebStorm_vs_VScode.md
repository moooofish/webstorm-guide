Visual Studio Code 作为编辑器界的*当红炸子鸡*，总会被开发人员拿来与 WebStorm 对比

**[WebStorm](https://www.jetbrains.com/webstorm/)** 是 JetBrains 公司开发的一个适用于 `JavaScript` 和相关技术的集成开发环境工具。它内置在开发中需要的各种配置和工具，提供开箱即用的功能。
它使用 Java 的 `Swing GUI` 创建用户界面，同时支持 `Windows`, `macOS` 和 `Linux` 平台。但是 WebStorm 闭源、收费，个人开发者使用收费 `6.9$`/月，这是一笔不小的开支；同时，WebStorm 对开发人员对电脑配置也有要求，电脑性能越好，WebStorm 的使用体验越棒。

**[Visual Studio Code](https://code.visualstudio.com/)**（下文简单称 VScode） 是微软公司推出的轻量级的代码编辑器，有海量的插件支持。
它基于 `Electron`、`node.js` 和 `TypeScript` 技术，同时支持 `Windows`, `macOS` 和 `Linux` 平台。VScode 背靠微软，依托开源、免费，受到
广大开发人员喜爱，市场占用率非常高。作为一个编辑器，VScode 本身主要提供目录工作区、全局搜索、代码调试和插件平台等主要功能，更多的功能需要开发人员安装各种插件去满足。

总的来说，WebStorm 与 Visual Studio Code 两款产品各有优缺点，萝卜青菜各有所爱，根据自己的喜好选择
